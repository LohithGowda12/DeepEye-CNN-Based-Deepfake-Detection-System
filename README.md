# DeepEye - CNN Based Deepfake Detection System

This project focuses on detecting deepfake images using a hybrid approach combining deep learning (CNNs) and machine learning classifiers.

## 🚀 Features
- CNN-based feature extraction (DenseNet, ResNet, Xception)
- Machine Learning classifiers (Random Forest, KNN, etc.)
- Explainable AI (Grad-CAM, SHAP)
- High accuracy deepfake detection

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- Scikit-learn
